# WebVault Build Notes

- Build a premium, sales-oriented web agency site in German.
- Keep the UI code-native; do not ship a static screenshot as the interface.
- Use Next.js, Tailwind v4, Framer Motion, shadcn/ui components, lucide icons, and Three.js for the hero scene.
- Design reference: `public/reference/webvault-concept.png`.
- Avoid generic SaaS card walls, gradient orb decoration, and cheap neon effects.
- Keep mobile layouts polished and check screenshots before handoff.
